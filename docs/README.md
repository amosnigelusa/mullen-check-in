# Pitch & paper — Mullen Library Patron Sign-In

Black-and-white theme, **Ubuntu** font. Three deliverables:

| File | What it is | How to use |
|---|---|---|
| `Mullen-CheckIn-Slides.pptx` | 15-slide pitch deck with speaker notes | Open in PowerPoint / Google Slides. Present with notes. |
| `Mullen-CheckIn-IEEE.docx` | IEEE paper as an editable **Word** document (two-column, full-width Table I) | Open in Word / Google Docs. Edit or **Save as PDF**. |
| `Mullen-CheckIn-IEEE.html` | Print-ready IEEE two-column paper | Open in Chrome → **Print → Save as PDF** (Letter, Default margins). |
| `Mullen-CheckIn-IEEE.tex` | IEEE LaTeX source (IEEEtran) | Upload to **Overleaf**, compile with **XeLaTeX** (for Ubuntu font) or pdfLaTeX (Times fallback). |

The Word, HTML, and LaTeX versions carry identical text; the deck tallies with the paper (same claims, same Table I).

## Notes
- **Ubuntu font:** the deck and paper request Ubuntu. If it isn't installed locally, PowerPoint substitutes a similar face; install the free Ubuntu font family for an exact match. The HTML paper loads Ubuntu from Google Fonts automatically (needs internet the first time).
- **The narrative** deliberately opens on the peak-hour queue and the forgotten hourly exit count, then frames the system as a zero-cost, working fix — sized to win a supervisor's top rating.
- The paper and final slide both note the project is **in active development**, with further features in progress.
- Timing figures are framed as informal observations; a controlled study is listed as future work (honest and defensible under questioning).
